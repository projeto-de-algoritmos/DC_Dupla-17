## Instalação

Para rodar as soluções, deve-se ter o C++ instalado.

## Uso 
Para rodar localmente os algoritmos, deve-se rodar o comando 

``` shell
    g++ -o teste main.cpp
```
e

``` shell
$ ./teste
```
e, por fim, colocar os inputs de teste disponibilizados.
